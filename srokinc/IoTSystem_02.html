IoT무선통신 개발

LINUX, WINDOWS OS 관련 COMPACT FLASH TYPE MEMORY 및 READER입니다.

  • 1

    COMPACT FLASH TYPE + NAND FLASH (AES적용)

  • 2

    NAND FLASH TO USB BRIDGE IC (AES적용)

  • 3

    USB, HUB, IC

  • 4

    HOST PC (LINUX OS, WINDOWS OS)

IoT무선통신 개발
  • 주요기술
  • 1. USB COMPACT FLASH CARD B/D 제작
    • USB HUB IC를 이용하여 COMPACT FLASH MAEMORY를 읽어들일수 있는 B/D 제작
  • 2. COMPACT FLASH CARD MEMORY B/D 제작
    • USB MEMORY IC 이용하여 COMPACT FLASH CARD MEMORY B/D 제작
  • 적용 가능한 ITEM
  • USB HUB IC를 이용한 MEMORY HUB 제작
  • USB HUB IC를 이용한 USB HUB 제작
  • 각종 MEMORY CARD 제작 SD MEMORY, COMPACT MEMORY 제작
  • USB 암호화 모듈 및 USB HUB 암호화 모듈
