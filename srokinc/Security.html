정보보안 무결성검증

OS 취약점 및 임베디드 디바이스를 공격하는 악성코드를 디지털 포렌식을 활용한 증거 수집 및 정보 보안 검증을 통해 분석, 저장 할 수 있는 도구를 연구하고 있습니다.

정보보안 무결성검증
  • 주요기술
  • 임베디드 시스템의 악성코드 리버스 엔지니어링
  • 디지털 포렌식을 활용한 정보 분석 도구 개발
  • 시스템 검증 도구 개발
  • 구축사례
  • 지능형 악성코드 탐지 및 차단기술
    • 개발환경 : Windows XP / 7 (32/64bit)
    • 개발언어 : C/C++, MFC
  • 네트워크 장비 보안 무결성 검증 프레임워크 개발기술
    • 개발환경 : VxWorks / Linux(Ubuntu)
    • 개발언어 : Mips & PowerPC Assembly, C/C++, Python
