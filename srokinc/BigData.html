AI분석 시각화
빅데이터 분석 및 시각화
  • 적용기술
  • GUI : html5, Ajax, Jquery, Css3
  • 지도 시각화 : Google Map api
  • 데이터 시각화 : OpenLayers V5
  • 모니터링 차트 (시각화 공통) : D3.js
  • 시각화 기법 및 기능
  • 빅데이터 시각화는 (1)정보 시각화, (2)시간시각화, (3)분포시각화, (4)관계시각화, (5)비교시각화, (6)공간시각화 기법을 사용하여 시각화한다.
  • 시각화 기법은 Html5를 기반으로 하며, Ajax를 이용한 비동기식 데이터 송수신으로 GUI를 통한 데이터 송수신 딜레이를 최소화 지역별, 시간대별 수집 및 분석/통계 데이터를 통하여 분석결과 관리와 수집데이터를 관리하고, React.js와 D3.js를 기반으로 하는 차트 UI를 제공함으로써 (1)정보/(2)시간/(4)관계/(5)비교시각화를 구현한다.
  • 프로토콜을 통한 센서별 HW동작 상태 모니터링을 관리 하고, Google Map api 와 OpenLayers v5 를 활용하여 GIS 상에 위험건수 및 위험수치에 대한 결과 및 통계지표를 제공 함으로써 (3)분포/(6)공간 시각화를 구현한다. 6가지의 시각화 기법을 적용하여 관리할 수 있는 시스템 관리 모듈을 제공한다.
