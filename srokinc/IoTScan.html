IoT-Scan
IoT-Scan 제품 목적 및 개요
  • 목적 및 개요
  • 사물인터넷(IoT, Internet of Things) 환경에서 발생 가능한 보안 위협에 대응하기 위한 IoT 기기 취약점 스캔도구 개발
  • 내부 네트워크에 연결되어 있는 기기가 공유기 / IP카메라 등 IoT 기기인지를 식별하고, 특정 정보를 스캔 할 수 있는 기능 구현
  • 공개된 취약점 존재 유무 점검 기능 구현(미라이, 쇼다운 등)
  • IP 범위 설정, 스캔할 정보 선택 등 스캔하기 위해서 필요한 설정들을 선택 할 수 있도록 GUI 형태로 개발
  • 스캔 결과는 Excel 포맷으로 산출
IoT-Scan 제품컨셉
  • 1 안정성이 보장되는 고속 스캔
  • 2 Windows 기반, 가벼운 엔진
  • 3 각 기능은 모듈화를 통해 확장성이 용이하고, 분리되어 각각 작동
  • 4 특정정보: 서비스, MAC 주소, IoT 기기 배너정보, 타입, 제조사, 모델명, 호스트네임, OS, Default ID/PW, UPNP
  • 5 취약점: Mirai, SShowDowN
  • 6 각 기능을 설정이 용이하게 GUI 형태로 개발
  • 7 스캔 결과 및 로그를 엑셀 형식으로 산출
IoT-Scan 주요기술
  • 주요기술
  • 1. USB COMPACT FLASH CARD B/D 제작
    • USB HUB IC를 이용하여 COMPACT FLASH MAEMORY를 읽어들일수 있는 B/D 제작
  • 2. COMPACT FLASH CARD MEMORY B/D 제작
    • USB MEMORY IC 이용하여 COMPACT FLASH CARD MEMORY B/D 제작
  • 적용 가능한 ITEM
  • USB HUB IC를 이용한 MEMORY HUB 제작
  • USB HUB IC를 이용한 USB HUB 제작
  • 각종 MEMORY CARD 제작 SD MEMORY, COMPACT MEMORY 제작
  • USB 암호화 모듈 및 USB HUB 암호화 모듈
